PREDICTIVE PARSING (LL(1)) - SAMPLE PROBLEM & TRACE

QUESTION:
Given the following Context-Free Grammar (CFG), construct the LL(1) Parsing Table and trace the validation of the input string: 'id * id + id'

Grammar:
E  -> T E'
E' -> + T E' | ε
T  -> F T'
T' -> * F T' | ε
F  -> ( E ) | id

-------------------------------------------------------------------------------
SOLUTION:

--------------------------------------------------
FIRST(E)  = { (, id }
FIRST(E') = { +, ε }
FIRST(T)  = { (, id }
FIRST(T') = { *, ε }
FIRST(F)  = { (, id }

2. FOLLOW SETS:
--------------------------------------------------
FOLLOW(E)  = { $, ) }
FOLLOW(E') = { $, ) }
FOLLOW(T)  = { +, $, ) }
FOLLOW(T') = { +, $, ) }
FOLLOW(F)  = { *, +, $, ) }

E' |           | E'-> +TE' |           |           | E' -> ε   | E' -> ε
